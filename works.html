Selected Work

Energy Forecasting Pipeline

I built a Python pipeline to clean, join, and analyze more than a million energy meter readings. After exploring the data to understand seasonality and outliers, I implemented baseline models and then experimented with ARIMA and gradient boosting. The final approach improved mean absolute error while keeping the code organized into reproducible steps with clear outputs and charts.

Python · Pandas · Statsmodels

Fryce POS

I designed a small point‑of‑sale desktop app for a campus pop‑up. It tracks inventory, sales, and daily summaries, and can export CSV/PDF for bookkeeping. The focus was on quick keyboard input, accurate totals, and easy setup so non‑technical users could operate it confidently.

Python · SQLite · Tkinter

Farm Tycoon Prototype

This game prototype explores the core loop of planting, harvesting, upgrading tools, and saving progress. I used Lua scripting to manage state, persistence, and simple UI feedback. The project taught me about economy balancing, player feedback, and how to structure code for iterative gameplay changes.

Roblox Studio · Lua
